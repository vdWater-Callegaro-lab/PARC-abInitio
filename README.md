# PARC-abInitio
Shared repository for the PARC ab initio case study, hosting scripts and analyses contributed by consortium partners.
   
<u>**Folder content:**</u>  
**1_Analyses**   
Contains the scripts provided by each partner to generate the input files for the Benchmark Dose (BMD) analysis (Quality Control).  
For DRomics, this folder also includes the code used to run the BMD analysis itself, and in some cases, scripts for the differential expression analysis.  
  
**2_BMD_Reporting_Template**  
Includes the template used to evaluate BMD analysis results.  
Running this script generates an HTML report to visually inspect your data.  
Details about the required input files are provided at the beginning of the script.  
  
**3_Manuscript_Figures**    
Contains the scripts used to generate the figures included in the manuscript.
